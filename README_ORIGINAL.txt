Send notifications to all your Android app user without paying fees as it does not use third-party servers.  

== Description ==
Easily send notifications to all of your android app user by using google Firebase Cloud Messaging key. No third-party service required. You can send custom message to all your android app user in a single click.


= Key Features =

Send Notifications to all of android app user :

* Firebase Cloud Messaging (FCM)

**Instant notifications.** Notifications appear as message alerts and even sound alerts.

= Who Is This Plugin For? =

This plugin is for mobile developers who do not want to develop their server-side back-end. Supporting push notifications is incredibly complicated. This plugin lets you focus on creating the apps, without the hassle. you can send custom notification to all of your android app user in a single click.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `send-fcm-notifications` folder to the `/wp-content/plugins/` directory
2. Activate Send FCM Notifications through the 'Plugins' menu in WordPress
3. Configure the plugin by going to the FCM Notifications menu that appears in your admin menu
4. In FCM settings under FCM Notifications menu put your google firebase cloud messaging access key (for all topic)
5. Follow the steps as detailed on FCM settings menu page to find your google firebase cloud messaging access key
